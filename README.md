# Snake RL - Double DQN

[English](README.en.md) | [Tiếng Việt](README.vi.md)

This repository trains an AI agent to play Snake with Deep Reinforcement Learning. It includes a Snake environment, a Double DQN agent, action masking, flood-fill safety features, training logs, checkpoint support, and an evaluation renderer.

Repo này huấn luyện AI chơi Snake bằng Deep Reinforcement Learning. Dự án gồm môi trường Snake, agent Double DQN, action masking, flood-fill để đánh giá vùng an toàn, log train, checkpoint và renderer để evaluate.

## Quick Start

```powershell
pip install torch numpy matplotlib pygame
python train.py
```

Evaluate after training:

```powershell
python evaluate.py
```
